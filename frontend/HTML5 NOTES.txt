HTML5
=====
HTML stands for Hypertext Markup Language.

It is widely used language on web to develop web pages and web applications.

It is developed by Burners Lee in late 1991.

HTML elements are represented in a tags so it is also known as tag based language.

HTML tag contains opening tag and closing tag.
ex:
	<html>  --> opening tag 

	</html> --> closing tag 

It is a case insensitive language.
ex:
	<html>  --> valid 
	<HTML>  --> valid 

HTML tag contains attribute and each attribute contains attribute name and attribute value.
ex:
	<font  color="red">

If a tag contains multiple attributes then each attribute must be seperated with space.
ex:
	<font  color="red"  size="10">

HTML document we need to save with .html or .html extension.

All HTML documents executed on browser window.
ex:
	Chrome 
	Mozilla
	Edge
	IE 
	Opera 
	UCBrowser 
	and etc.

We will develop HTML document in following editors.
ex:
	Notepad 
	Notepad++
	Atom
	Sublime 
	VSC 
	DreamViewer 
	and etc.

A simple HTML document is known as component.

HTML is used to develop client side components.

HTML2.0 was released to the public in the year of 1995.

HTML4.0 was released in the year of 1999.

HTML5 which is extension of HTML4.0 was released in the year of 2005.

The main objective of HTML5 is to build leight weight components.

HTML5 is also known as Advanced Hypertext Markup Language.

HTML comments
==============
Comments  are created for documentation purpose.

Comments are used to improve readability of our code.

Comments will not appear in output/browser window.

We can declare HTML comment as follow.

syntax:
------
	<!--    comment here  --> 

ex:
---
<!DOCTYPE html>
<html>
	<head>
		<!-- title -->
		<title>IHUB TALENT</title>

		<!-- favicon -->
		<link rel="icon" href="icon.png"/>
	</head>
	<body>
		<!-- custom msg -->
		Welcome to HTML classes
	</body>
</html>

Metadata
=========
A <meta> tag is used to declare metadata of a document.

Here metadata means data of a data.

Metadata is used by search engine, browser window and other web services.

To declare metadata we need to use <meta> tag.

A <meta> tag must declare inside <head> tag.

A <meta> tag is used to declare following things.

1) Description 

2) Author 

3) Keywords

4) Viewport

5) Refresh 

6) copyright 

and etc.

Q) Types of Tags in HTML?

We have two types of tags in HTML.

1) Paired  Tags / Container Tags
--------------------------------
Tags which contain opening tag and closing tag are called paired tags.
ex:
 	<h1>, <p>, <body>, <head> , <title> and etc.

2) Unpaired Tags / Empty Tags 
----------------------------
Tags which contain opening tag and does not have any closing tag are called unparied tags.
ex:
	<meta>, <br> , <hr> and etc.

HTML Colors
============
There are four ways to define a color in HTML.

1) Valid color name 

2) Hexa value 

3) RGB value 

4) HSL value 


1) Valid color name 
-------------------
<!DOCTYPE html>
<html>
	<head>
		<title>MyPage!</title>
	</head>
	<body bgcolor="cyan">
				
			<font color="brown" size="30">
				Welcome to HTML class
			</font>
	</body>
</html>

2) Hexa value 
-----------------
Hexa value starts with '#' symbol followed by 6 characters.

ex:
<!DOCTYPE html>
<html>
	<head>
		<title>MyPage!</title>
	</head>
	<body bgcolor="#FFFF00">
				
			<font color="#FF0000" size="30">
				Welcome to HTML class
			</font>
	</body>
</html>

3) RGB value
-------------
RGB stands Red Green Blue.

RGB value having range from 0 to 255.

HTML does not support RGB value directly.

ex:
<!DOCTYPE html>
<html>
	<head>
		<title>MyPage!</title>
	</head>
	<body bgcolor="rgb(0,0,255)">
				
			<font color="rgb(0,255,0)" size="30">
				Welcome to HTML class
			</font>
	</body>
</html>

4) HSL value 
---------------
HSL stands for Hue Saturation and Lightness.

HTML does not support HSL value directly.

ex:

<!DOCTYPE html>
<html>
	<head>
		<title>MyPage!</title>
	</head>
	<body bgcolor="hsl(240, 100%, 50%)">
				
			<font color="hsl(120, 100%, 50%)" size="30">
				Welcome to HTML class
			</font>
	</body>
</html>

Q) What is the difference between Tag and Element?

Tag
=====
HTML tag contains '<' symbol and '>' symbol.
ex:
	<h1>,<br>,<p>,<div> and etc.

Element
========
HTML element contains opening tag , children  and closing tag.
ex:
	<h1> This is heading tag </h1>
	<p> This is paragraph tag </p>
	and etc.

Graphics in HTML
================
There are two ways to draw the graphics in html.

1) SVG 

2) CANVAS 

1) SVG 
--------
SVG stands for Scalable Vector Graphics.

HTML5 uses SVG technology to derived the graphics.

SVG is used to draw two dimensional vector graphics.

W3C (World Wide Web Consortium) prefers SVG technology to draw vector graphics.

A <svg> is a container tag.

A <svg> tag contains number of methods to display circle, rectangle ,polygon, graphic images and etc.


ex:
----
<!DOCTYPE html>
<html>
	<head>
		<title>MyPage!</title>
	</head>
	<body>
		<svg style="border:2px solid black">
			
			<circle cx="150"  cy="75"  r="50"  
				fill="#FFFF00" stroke="#FF0000" stroke-width="5px"> 
			
		</svg>
	</body>
</html>

2) CANVAS
---------
It is used to draw two dimensional vector graphics via javascript.

A <canvas> is a container tag.

A <canvas> tag contains number of methods to draw lines, circle, rectangle, graphic images and etc.

ex:1
----
<!DOCTYPE html>
<html>
	<head>
		<title>MyPage!</title>
	</head>
	<body>
		<canvas style="border:2px solid black" id="myId">
		
		</canvas>
		
		<script>
			var context=document.getElementById("myId");
			var c=context.getContext("2d");
			c.moveTo(0,0);
			c.lineTo(800,400);
			c.stroke();
		</script>
		
	</body>
</html>

ex:
