CSS3
=====
CSS stands for Cascading Styles Sheet.

It is used to apply the styles on elements.

CSS3 was introduced in the year of 2001.

The main objective of CSS3 are.

1) To set the positioning of an element.

2) To apply the styles on elements to describe how an element should look like.

3) To perform some sort of animations.

ex:
----
<!DOCTYPE html>
<html>
	<head>
		<style type="text/css">
			-
			- // CSS properties 
			-
		</style>	
	</head>
	<body>
		-
		-
		-	
	</body>
</html>

ex:
----
<!DOCTYPE html>
<html>
	<head>
		<title>MyPage!</title>
		<style type="text/css">
			h1
			{
				color:blue;
				background-color:cyan;
				text-align:center;
			}
		</style>
	</head>
	<body>
			<h1>This is Heading Tag</h1>
	</body>
</html>

Note:
-----
	Here styles cascading from head to body.

CSS syntax
===========
CSS rule set contains selector and declaration block.
ex:
				Declaration block
					|
	  |--------------------------------------------------|
	h1{color:blue;background-color:cyan;text-align:center}
	|
	selector 	

Selector indicates , to which element we need to apply the styles.

Declaration block contains collection of CSS properties.

Each CSS property seperated with semicolon(;).

Each property contains property name and property value seperated with colon(:).


Advantages of CSS
=================
1) It is easy to learn and easy to use.

2) It saves lot of development time. 

3) It supported by all major browsers.

4) Flexibility 

5) It supports global change.

6) Performance is faster

Disadvantages of CSS
====================
1) Fragmentation 

2) Need to update all CSS versions.

Types of CSS
============
We have three types of CSS.

1) Inline CSS 

2) Internal CSS 

3) External CSS  


1) Inline CSS
--------------
Inline CSS is used to apply unique style on single element.

Using "style" attribute we can achieve inline CSS.

ex:

<!DOCTYPE html>
<html>
	<head>
		<title>MyPage!</title>
		
	</head>
	<body>
		<h1 style="color:blue;background-color:cyan;">This is Heading Tag</h1>
	</body>
</html>

2) Internal CSS 
---------------
If we want to apply unique styles on a single web page.

Using <style> tag is used to achieve internal CSS.

A <style> tag  we need to declare inside <head> tag.

ex:
--
<!DOCTYPE html>
<html>
	<head>
		<title>MyPage!</title>
		<style>
			h1
			{
				color:blue;
				text-align:center;
				background-color:cyan;
			}
		</style>
	</head>
	<body>
			<h1>This is Heading Tag</h1>
			
			<h1>This is Heading Tag</h1>

			<h1>This is Heading Tag</h1>	
	</body>
</html>

3) External CSS 
------------------
If we want apply unique styles on entire website.

In external CSS we will create .html file and .css file.

A .html file contains HTML code and .css file contains CSS code.

But we can't execute .css file directly on browser window.

We need to embed .css file in  .html file by using a <link> tag.

ex:
---

index.html 
----------
<!DOCTYPE html>
<html>
	<head>
		<title>MyPage!</title>
		
		<!-- add external CSS -->
		<link rel="stylesheet" type="text/css" href="mystyles.css">
		
	</head>
	<body>
		<h1>This is Heading Tag </h1>
		<p> This is Paragraph Tag </p>
	</body>
</html>

mystyles.css
------------
h1
{
	color:blue;
	text-align:center;
}
p
{
	background:#00ff00;
	font-size:40px;
}

3) margin-bottom 

4) margin-left

CSS overflow property
=======================

The overflow property specifies what should happen if content overflow.

This property spcifies weither to clip content or to add scrollbars  when an element content is too big to fit in a specified area.

Note:
---------
The overflow property works for block elements with  a specified height.

value		Description
---------	-----------------
visible		The overflow is not clipped.It rendered outside the element 's box and 
		it is default value.

hidden		the overflow is clipped  and rest of the content will be invisible.

scroll		The overflow is clipped,but a scroll-bar is added to see the rest of the 
		content .

auto		The overflow is clipped,a scroll-bar should be added to the rest of the 
		content.

We have five types of selectors in CSS.

1) Element selector 

2) Id selector 

3) Class selector 

4) Group selector 

5) Universal selector 


1) Element selector 
-------------------
The element selector selects HTML elements based on element name.


ex:
---
<!DOCTYPE html>
<html>
	<head>
		<title>MyPage!</title>
		<style>
			h1
			{
				color:blue;
				text-align:center;
				background-color:yellow;
			}
			
		</style>
	</head>

	</body>

2) Id selector 
---------------
The id selector uses the id attribute of an HTML element to select a specific element.

The id of an element is unique within a page, so the id selector is used to select one unique element.

To select an element with a specific id, write a hash (#) character, followed by the id of the element.


ex:
--
<!DOCTYPE html>
<html>
	<head>
		<title>MyPage!</title>
		<style>
			#id1
			{
				color:blue;
				text-align:center;
				background-color:yellow;
			}
			#id2
			{
				font-style:italic;
				font-size:50px;
				text-align:center;
				color:green;
			}
			
		</style>
	</head>
	<body>
			<h1 id="id1"> This is Heading Tag 1</h1>
			
			<h1 id="id2"> This is Heading Tag 2</h1>	
	</body>
</html>

CSS Display 
============
The display property spcifies the display behaviour of an element.

syntax
--------
	display: value;

value		description
---------	-------------------
none		The element is completely removed.

inline 		displays an element as on inline element.
		Any height and width properties will not effect.

block		Displays an element as block element.
		IT starts on a new line and takes up the whole width.

inline-block	displays an element as an inline-level container.
		The element itself is formatting  as an inline element.
		but we can apply height and width values.

inherit		Inherits this property from its parent element.

CSS tranform
==============
CSS tranform property allows use to move ,rotate  or skew elements.

CSS tranform property contains following transformation methods.

ex:
	translate()
	rotate()
	scaleX()
	scaleY()
	skewX()
	skewY()
	skew()


transform: translate()
-----------------------
The translate() method moves  an element from its current position to the parameters given by the X-axis and the Y-axis.
ex:


ex:
---
<!DOCTYPE html>
<html>
<head>
	<title>facebook</title>

		#box
		{
			width: 200px;
			height: 200px;

CSS position property
=====================
The position property specifies the type of positioning method used for an element (static, relative, absolute, fixed, or sticky).

The following are the list of values to position property.

static 
---------------
	It is default value. Elements render in order, as they appear in the document flow 	

absolute 
--------------
	The element is positioned relative to its first positioned (not static) ancestor element 	
fixed 	
-----------
	The element is positioned relative to the browser window 	
relative 
-------------
	The element is positioned relative to its normal position, so "left:20px" adds 20 pixels 	to the element's LEFT position.
 	
sticky 	
-----------
	The element is positioned based on the user's scroll position 


css position property mandatory should have following properties.
i)left
ii)right
iii)top
iv)bottom

CSS Google Fonts
=================
If we do not want to use any of the standard fonts in HTML, you can use Google Fonts.

Google Fonts are free to use, and have more than 1000 fonts to choose.


To use any google fonts we need to use below url.
ex:
	https://fonts.google.com/

We can use google font in two ways.

1) <link> tag 

2) @import 


1) <link> tag 
---------------
<!DOCTYPE html>
<html>
	<head>
		<title>MyPage!</title>
		<link href="https://fonts.googleapis.com/css2?family=Merienda:wght@300..900&display=swap" rel="stylesheet">
		<style>
			h1
			{
				font-family: "Merienda", cursive;
			}
			
		</style>
	</head>
	<body>
		<h1> This is Heading tag </h1>
	</body>
</html>



2) @import
----------
<!DOCTYPE html>
<html>
	<head>
		<title>MyPage!</title>
		<style>
		@import url('https://fonts.googleapis.com/css2?family=Merienda:wght@300..900&display=swap');
			h1
			{
				font-family: "Merienda", cursive;
			}
			
		</style>
	</head>
	<body>
		<h1> This is Heading tag </h1>
	</body>
</html>
